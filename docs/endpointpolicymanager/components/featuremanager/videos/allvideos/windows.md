# Feature Manager For Windows

Add or Remove Windows Features and Optional Features, like SMB 1.0, XPS Viewer, RSAT tools or
anything else. Use Group Policy, Netwrix Endpoint Policy Manager (formerly PolicyPak) Cloud or MDM
to do it.
